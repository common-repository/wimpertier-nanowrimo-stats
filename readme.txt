Widget with the stats from a NaNoWriMo writer
== Description ==
* Daily stats about the written words from a NaNoWriMo writer.
* How many words has a writer to write to reach 50 000 words.
* When a writer has reached 50 000 words.
== Installation ==
1. Upload 'wtnnwm-nanowrimostats' to the '/wp-content/plugins' directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the widget to the 'Widgets Area' on the 'Widget' screen
4. Add a NaNoWriMo Username and set the widget options
